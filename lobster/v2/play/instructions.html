FAQs:

  • Q: Why isn't my game saving?

    A: Some systems clear local browser data periodically. If your system has a persistent folder, you can configure your browser to save data in that folder.
    For example, using Firefox on Windows:

    1. Navigate to your persistent folder. We use the Documents folder in this example.
    2. Right-click in the Documents folder and create a new folder called myffprofile .
    3. Right-click in the Documents folder again. Pick New > Shortcut.
      A popup window should appear asking you to choose the target.
    4. Copy/paste the following:
      "C:\Program Files\Mozilla Firefox\firefox.exe" -profile "%USERPROFILE%\Documents\myffprofile" -no-remote https://python-can-define-radio.github.io/lobster/
    5. Name the shortcut Firefox_persistent. Click "Finish".

  • Q: What is this game?

    A: Radio Direction Finding (DFing) is the process of locating a radio transmitter. This can be done by simply measuring the received power and observing where it is stronger or weaker, but it is often more effective to determine the Angle of Arrival (AoA), or azimuth angle. This technique is used by hobbyists, cell phone companies, law enforcement, and military.

    Using a military context, this game provides the opportunity to experience some of the challenges of DFing without equipment.
